# SSH Investigation Dashboard

## Purpose

Provides a centralized view for analyzing SSH attack activity and supporting alert investigations.

## Key Visualizations

- Failed login attempts over time
- Successful authentication events
- Top attacking IP addresses
- Command execution activity

## Usage

Used during investigations to:
- Identify attack patterns
- Pivot on suspicious IPs
- Review attacker behavior post-compromise

## Outcome

Reduces investigation time and provides quick situational awareness during active alerts.
